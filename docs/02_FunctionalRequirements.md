### Functional Requirements

- The system shall accept long text documents as input via an API.
- The system shall generate a concise, human-readable summary of each document.
- The system shall extract key information and represent it in a structured format.
- The system shall return both summary and structured data in machine-readable formats (e.g., JSON).
- The system shall support integration with downstream systems through well-defined APIs.
- The system shall handle documents of varying length and structure.
